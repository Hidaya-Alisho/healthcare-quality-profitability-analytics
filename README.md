# Healthcare Quality & Profitability Analytics

## Overview
This project analyzes the relationship between CMS hospital quality scores and financial performance across rural and urban hospitals in the United States. The goal is to understand whether quality ratings correlate with profitability and to identify regional and hospital-type patterns.

## Business Questions
- Do CMS quality scores correlate with hospital profitability?
- Does profit tend to increase/decrease with higher CMS scores?
- How do results differ by state, hospital type (rural vs urban), and specialty?

## Methods
- Data cleaning and transformation
- Exploratory data analysis (EDA) and visualization

## Tools
- Python (pandas, numpy, matplotlib, seaborn, scikit-learn)
- Google Colab
